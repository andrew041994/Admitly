# Admitly

Premium event ticketing platform for Guyana.

## Stack
- Backend: FastAPI
- ORM: SQLAlchemy
- Migrations: Alembic
- Database: PostgreSQL
- Admin frontend foundation: React + TypeScript + Vite (`admin/`)
- Mobile app foundation: Expo + React Native (`mobile/`)

## Planned apps
- Backend API
- Organizer/admin web frontend
- Mobile app
